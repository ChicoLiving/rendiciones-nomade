Rendiciones Nomade

Esta es una versión de ejemplo de la aplicación.
Incluye OCR, exportación a Excel y soporte para uso web desde móvil o escritorio.

Instrucciones básicas:
1. Ejecuta app.py con Python 3.8+ y las dependencias instaladas.
2. Accede desde tu navegador en http://localhost:8501
3. Sube archivos de boletas o facturas.
4. Exporta a Excel desde el botón en la parte inferior.
